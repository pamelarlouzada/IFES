Operadores relacionais

Os operadores são:

Exemplos de aplicação:

  //Operadores relacionais (demonstração)
  x = 10
  y = 20
  z = "10"
  
  console.log("x =", x, "y =", y, "z =", z)
  console.log("x == z: ", x == z)
  console.log("x != z: ", x != z)
  console.log("x === z: ", x === z)
  console.log("x !==z: ", x !==z)
  console.log("x < y : ", x < y )
  console.log("x > y : ", x > y )
  console.log("x <= y : ", x <= y )
  console.log("x >= y : ", x >= y )

  ----------------------------------------------
  x = 10 y = 20 z = 10
  x == z:  true
  x != z:  false
  x === z:  false
  x !==z:  true
  x < y :  true
  x > y :  false
  x <= y :  true
  x >= y :  false
