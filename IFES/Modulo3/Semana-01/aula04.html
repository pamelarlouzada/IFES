Variáveis

As variáveis oferecem uma maneira comveniente de armazenar dados.

Uma variável é um nome dado a um local de memória.

Quando uma variável é declada, de acordo com o tipo de dado que armazena, algum espaço é reservado para ela na memória do sistema.

O javaScript pode lidar com diferentes tipos de dados (Data Type).

Um nome de uma variável pode conter letras, números, underscore e cifrão ($).

O primeiro caractere de um nome de uma variável não pode ser um número.

Palavras chaves não podem ser usadas como variável.

Para atribuirvalores a variáveis, usamos o operador de atribuiçãoo (=). A sintaxe geral de atribuição de valores a variáveis é:

  <nome da variável> = <valor>

Não é necessário declarar o tipo da variável no momento da attribuição.

Atribuição multipla

A atribuição múltipla de variáveis pode ser feita da seguinte forma: 

  <variável 1> = <variável 2> =… <variável n> = <valor>
  a = b = c = d = 100;

Tipos de dados

JavaScript tem 7 tipos de dados padrões:

  • Boolean: true e false
  • null
  • undefined
  • Number: 42 ou 3.14159
  • String: "João" ou 'João'
  • Symbol
  • Object

Mostrar conteúdo de variáveis

A maneira mais simples de exibir variáveis é usando a função console.log. Sintaxe geral:

console.log (<variável>)
console.log (<variável 1>, ..., <variável n>)

Constantes e variáveis podem ser combinadas em uma única instrução console.log, separando constantes e/ou variáveis strings usando vírgulas.

nome = "Maria";
idade = 32;
console.log("Nome:", nome, "Idade:", idade)
----------------------------------------------
Nome: Maria Idade: 31

Se quiser exibir algo em uma nova linha, você pode incluir a sequência de caracteres \n dentro da instrução console.log.

nome = "Maria";
idade = 32;
console.log("\nNome:", nome, "\nIdade:", idade)
----------------------------------------------
Nome: Maria
Idade: 32

Declaração de Variáveis

Você pode declarar uma variável de três formas:

  • Com a palavra chave var. Por exemplo, var x = 41. Esta sintaxe pode ser usada para declarar tanto variáveis locais como variáveis globais.
  • Por simples adição de valor. Por exemplo, y = 42. Isso declara uma variável global. Essa declaração gera um aviso de advertência no JavaScript. Você não deve usar essa variante.
  • Com a palavra chave let. Por exemplo, let z = 43. Essa sintaxe pode ser usada para declarar uma variável local de escopo de bloco.
var x = 41
y = 42
let z = 43
  // Demonstração de escopo de variável   <
  if (true) {
    var x = 41;   // variável de escopo global ou local
    y = 42;       // variável de escopo global
    let z = 43;   // variável de escopo de bloco
  }

  console.log(x)
  console.log(y)
  console.log(z)
  ----------------------------------------------
  ReferenceError: z is not defined
  • A declaração const cria uma variável cujo o valor é fixo, ou seja, uma constante somente leitura. Isso não significa que o valor é imutável, apenas que a variável constante não pode ser alterada ou retribuída.
  const nome1 = valor1;
  const PI = 3.14;

  // define PI como uma constante e lhe atribui o valor 3.14
  const PI = 3.14;
  
  // tentar atribuir novo valor para constante (TypeError)
  PI = 20;
  
  // a variável PI possui o valor 3.14
  console.log("meu número favorito é: ", PI);
  
  // tentar redeclarar a constante emite um erro:
  // SyntaxError: Identifier 'PI' has already been declared
  //const PI = 20;
  
  // o nome PI está reservado para a constante acima, logo irá falhar
  var PI = 20;
  
  // isso também vai falhar
  let PI = 20;
  
  // const deve ser inicializada:
  // SyntaxError: missing = in const declaration
  const AUX;
