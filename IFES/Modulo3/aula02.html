Introdução

Verifique se o Node instalado digitando node -v no terminal (se não tiver instalado, faça a nstalação)

Crie um arquivo aula02.js

Digite console.log("Olá Mundo") no arquivo aula02.js

Certifique-se que o diretório do terminal esteja na pasta correta

Rode o comando node aula02.js no terminal. Deverá aparecer "Olá Mudndo"

  console.log("Olá Mundo")

  ----------------------------------------------
  $ node aula02.js
  Olá Mundo
