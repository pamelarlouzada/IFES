Histórico da linguaguem

JavaScript é uma Linguagem de Programação, criada por Brendan Eich, a pedido da empresa Netscape, em meados de 1995. No início, o JavaScript foi batizado com outro nome: LiveScript. No entanto, a Netscape não ficou sozinha com o desenvolvimento do JavaScript.

Características da linguaguem

É uma linguagem interpretada, Isso significa que não é preciso um compilador para rodar os programas; o código é executado de cima para baixo e o resultado da interpretação do código é imediatamente retornado (no js, os navegadores de internet conseguem fazer essa interpretação)

É uma linguagem multiparadigma, suportando diferents estilos de programação como procedural, orientado a objetos, etc, ou seja, é possível programar em diversas perspectivas e abordagem de problema

É uma linguagem de propósito geral, ou seja, para ser utilizada em diferentes finalidades: para front-end, back-end (Node), programação em dispositivos móveis (React Native), elaboração de jogos (PalyCanvas & Phaser)

É uma linguagem leve, fácil, organizada e intuitiva. EM comparação com outras linguagens de programação, o javascript não é difícil de ser aprendido. Por ser uma linguagem popular, existe uma comunidade bem atuante facilitando a busca de soluções para diferentes problemas

É uma linguagem muito utilizada. É a linguagem mais utilizada em 2022 segundo pesquisa do Stack Overflow.

Como a web funciona

Website

HTML significa "Hyper Text Markup Language", que pode ser traduzido mais livremente como "linguagem de marcação de hipertexto"

CSS siginifica "Cascading Style Sheet", algo como "Folha de Estilos em Cascata", tem a finalidade de estilizar uma página web

Javascript é a linguagem de programação que deixa a página com uma funcionalidade mais dinâmica, dando movimento para a página e permitindo o rápido acesso, processamento e transformação de dados enviados e recebidos
