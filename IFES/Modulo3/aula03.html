Palavras Chaves

Palavras Chaves são palavras reservadas que não podem ser usadas como nomes de identificadores. O ambiente de desenvolvimento destaca essas palavras para auxílio.

Instruções

Uma instrução é usada para realizar uma tarefa ou um grupo de tarefas

Uma tarefa pode ser qualquer coisa com realizar operações aritiméticas, verificar se dois números são iguais, etc. Geralmente, uma instrução está presente em uma linha. 

  1   pais = "Brasil";                      | instrução de atribuição
  2   console.log("Exemplo de Saída");      | instrução de saída
  3   produto = 5.86 * 82.41;               | instrução de multiplicação
  4   vendido = true;                       | instrução de atribuição

  ** Instrução se refere a finalidade da linha de comando

O ( ; ) é utilizado cmo delimitador de instrução, porem, em javascript nós temos o ASI (Automatic Semicolon Insertion), ele é responsável por inserir automaticamente o ( ; ) em determinadas situações, fazendo com que cada quebra de linha seja adicionado o ; automaticamente. No entando, existem casos específicos que podemso ter problemas com essas inserções automáticas.

Blocos de Códigos

Um bloco de código é uma coleção de algumas linhas de instruções. Um bloco é denotado por abre e fecha chaves

  for (i = 1; i <= 10; i++) {
    console.log("i = ", i);
  }

Comentários

Os comentários são linhas ignoradas pelo interpretador e usada para marcar, descrever ou explicar um trecho do código

Existem comentários de uma única linha (//), ou em blocos (/* */)

  // comentário em uma linha

  /* 
     isso é um comentário longo
     de multiplas linhas
   */
