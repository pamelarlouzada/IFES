Criando um Array

São coleções de dados ordenadas por um valor indexado. Neste caso específico vamos abordar objetos do tipo Array.

  var emp = ['João', 'Maria', 'Augusto'];
  console.log(emp)
  --------------------------------------------
  [ 'João', 'Maria', 'Augusto' ]

  var emp = ['João', 'Maria', 'Augusto'];
  console.log(emp)

  var emp = [];
  emp[0] = 15;
  emp[1] = 50;
  emp[2] = 70;
  
  console.log(emp)
  --------------------------------------------
  [ 'João', 'Maria', 'Augusto' ]
  [ 15, 50, 70 ]

Referenciando um Array

  var emp = ['João', 'Maria', 'Augusto'];
  console.log(emp[0])
  console.log(emp[1])
  console.log(emp[2])
  --------------------------------------------
  João
  Maria
  Augusto

Iterando um Array

São coleções de dados ordenadas por um valor indexado. Neste caso específico vamos abordar objetos do tipo Array.

  var emp = ['João', 'Maria', 'Augusto'];
 
  for (var i = 0; i < emp.length; i++){
    console.log(emp[i])
  }
  --------------------------------------------
  João
  Maria
  Augusto
