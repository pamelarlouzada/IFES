Resumo

Crie rótulos de formulário simples que flutuem sobre seus campos de entrada. Para isso, faça uma <div class="form-floating"> depois acrescente um <input class="form-control"> seguido de uma label

Se colocar um [placeholder=" "] com um espaço, o título vai para o meio do input


Defina um [value="test@example.com"]


Acrescente validações com a class .is-invalid no input

Textareas

Use uma <textarea class="form-floating"> depois acrescente uma <div class="form-control"> e complete uma uma <label>

Select

Use uma <textarea class="form-floating"> depois acrescente uma <select class="form-select"> e complete uma uma <label>

Não funciona com size
