Resumo

.form-control é usado para input <input class="form-control" type="email" id="email">

.form-label é usado em label <label class="form-label" for="email">E-mail</label>

.form-text é usado em textos de ajuda/erro <div class="form-text">Seu email não será compartilhado</div>

.form-chek deve ser usado na div do checkbox (ele dá o espaçamento entre o checkbox e o texto) <div class="mb-3 form-check">

.form-chek-input usado no input <input class="form-check-input" type="checkbox" id="lembrar">

.form-chek-label usado na label <label class="form-check-label" for="lembrar">Lembrar senha</label>

.btn .btn-primary usado nos botões <button class="btn btn-primary" type="submit">Entrar</button>

Atributos combinados com Bootstrap

O bootstrap também possui formatação para elementos do HTML quando uma classe está declarada. Esses atributos são: [disable] [readonly] [placeholder=""]

O [readonly] quando combinado com o atributo [value=''], permite que o texto seja copiado

Sizing

Defina alturas usando classes como .form-control-lg e .form-control-sm.

Disabled e Readonly

Adicione o atributo disabled em um input para dar a ela uma aparência esmaecida e remover eventos de ponteiro do mouse.

Adicione o atributo readonly em um input para impedir a modificação do valor da entrada.

Readonly plain text

Se você deseja ter <input readonly> em seu formulário estilizados como texto simples, use a class .form-control-plaintext para remover o estilo padrão do campo de formulário e preservar a margem e o preenchimento corretos.

Outros forms

Veja a aplicação da classe .form-control-{color} em outros inputs: Para arquivos <input type="file" name="" id="" class="form-control"> Para cores <input type="color" name="" id="" class="form-control">
