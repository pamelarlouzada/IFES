Checkbox

.form-check dentro da div: <div class="form-check">

.form-check-input no input: <input class="form-check-input" type="checkbox" id="ex01">

.form-check-label na label: <label class="form-check-label" for="ex01"> Exemplo de checkbox

Radio

Basta mudar o type para radio <input class="form-check-input" type="radio" name="flexRadioDefault" id="ex04">

Lebrando que para selecionar uma opção, os inputs precisam ter o mesmo name. Também é preciso definir um value par ser enviado ao servidor

Switches

.form-switch dentro da div: <div class="form-check form-switch">

Atributos

[disabled] Desabilita o campo <input class="form-check-input" type="checkbox" value="" id="ex02" disabled>

[disabled] [checked] Desabilita o campo marcado <input class="form-check-input" type="checkbox" value="" id="ex03" checked disabled>

Design - Inline

.form-check .form-check-inline para deixar os inputs em linha <div class="form-check form-check-inline">

Design - Reverse

.form-check .form-check-reverse para deixar alinhadoa direita <div class="form-check form-check-reverse">

Toggle buttons

Diferentes variantes de .btn, como nos vários estilos delineados, são suportadas.
