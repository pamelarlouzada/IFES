Resumo

Por padrão, os elementos de um formulário são organizados de forma vertical, uma vez que os elementos ocupam 100% do elemento pai e tem o display="block". Use as classes.row e col. 

  <div class="row">
    <div class="col">
      <input type="text" class="form-control" placeholder="First name">
    </div>
    <div class="col">
      <input type="text" class="form-control" placeholder="Last name">
    </div>
  </div>

Horizontal form

A classe que permirte defir a quantidade de colunas para o <label> é a .col-form-label seguida da quantidas de colunas desejadas .col-2.

Radios 
  <form>
    <div class="row mb-3">
      <label for="ex01" class="col-sm-2 col-form-label">Email</label>
      <div class="col-sm-10">
        <input type="email" class="form-control" id="ex01">
      </div>
    </div>

    <div class="row mb-3">
      <label for="ex02" class="col-sm-2 col-form-label">Password</label>
      <div class="col-sm-10">

    </div>

    <fieldset class="row mb-3">

    <button type="submit" class="btn btn-primary">Sign in</button>
    </form>

Sizing

Adicione .col-form-label-{sm lg} ao <label>

O dimensionamento dos elementos individuais do grupo de entrada não é suportado.

  <div class="row mb-3">
    <label for="ex06" class="col-sm-2 col-form-label col-form-label-sm">Email</label>
    <div class="col-sm-10">
      <input type="email" class="form-control form-control-sm" id="ex06" placeholder="col-form-label-sm">
    </div>
  </div>

  <div class="row mb-3">
    <label for="ex07" class="col-sm-2 col-form-label">Email</label>
    <div class="col-sm-10">
      <input type="email" class="form-control" id="ex07" placeholder="col-form-label">
    </div>
  </div>
  <div class="row">
    <label for="ex08" class="col-sm-2 col-form-label col-form-label-lg">Email</label>
    <div class="col-sm-10">
      <input type="email" class="form-control form-control-lg" id="ex08" placeholder="col-form-label-lg">
    </div>
  </div>
