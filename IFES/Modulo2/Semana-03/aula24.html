Resumo

.needs-validation: colocar no elemtno <form> e para ativar essa class é preciso acrescentar o atributo [novalidation] no <form> isso dasativara as validações padrões.

Crie uma div com a class.invalid-feedback ou .valid-feedback após o input para escrever uma mensagem de erro ou validação. Essa validação é feira apenas no navegador, então qualquer usuário consegue burlar facilmente. Por isso as validações precisam ser feitas no servidor

Classes de servidor: .is-valid .is-invalid
